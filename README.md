# Instructions

# Dev environment

1. Clone the project
2. Copy `.env.example` to `.env` and fill in the values
3. Execute `yarn install`
4. Set up the DB `docker-compose up -d`
5. start the project `yarn start:dev` will run on `localhost:3000`
